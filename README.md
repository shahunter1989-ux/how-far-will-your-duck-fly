# How Far Will Your Duck Flap?

A simple mobile browser game where tapping controls a duck's altitude in a top-down pond view.

## Rules

- Ascend: 4 or more taps per second.
- Descend: 1 to 3 taps per second.
- Fast fall: no taps for 0.75 seconds.
- Game over: altitude reaches zero.
- Score: increases over time based on current altitude.

Open `index.html` in a browser to play.
